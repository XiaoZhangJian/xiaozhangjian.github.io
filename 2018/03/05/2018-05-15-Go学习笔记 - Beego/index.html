    限定使用方式请求的类型：

    • * : 允许任何请求方式
    • get : 只允许get
    • post : 只允许post
    • put : 只允许put
    • delete : 只允许delete

    内置基础类型:

    • bool
      布尔值类型为bool ，值是true或false，默认false
    • 数值类型
      (u)int,(u)int8,(u)int16,(u)int32,(u)int64,uintptr
      整数类型有无符号和带符号两种。Go同时支持int 和 uint,有符号的整数分为两类：一类为规定长度 int8,int16,int32,int64。不规定长度的长度是根据运行系统来规定的。
